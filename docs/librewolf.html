LibreWolf — Firefox, Hardened and De-Telemetrized

Open-source browser based on Firefox with telemetry removed and privacy/security preferences pre-tuned.

Updated: 2025-09-10

What LibreWolf changes

  • No telemetry and no experiments.
  • Hardened defaults (cookies, tracking protection, referrers, etc.).
  • Search & new-tab tweaks oriented toward privacy.
  • Regularly rebased on Firefox ESR/stable with community-maintained patches.

Goal: Firefox compatibility with stronger privacy out-of-the-box.

Install

  • Linux/macOS/Windows: grab builds from the official LibreWolf website or supported repos.
  • Profile separation works like Firefox—keep a clean profile for sensitive work.

Recommended add-ons

  • uBlock Origin — network-level blocking.
  • ClearURLs — strip tracking params from links.
  • LocalCDN — emulate common CDNs locally.
  • (Advanced) NoScript — script control, if you like tuning.

Tips to avoid breakage

  • Use per-site permissions (lock icon) instead of weakening global prefs.
  • If a site demands cross-site cookies, prefer a Temporary Container or a separate profile.
  • For video DRM sites, consider a separate vanilla Firefox profile for that task only.
