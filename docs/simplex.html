SimpleX Chat — Messaging Without Identifiers

No usernames, no phone numbers, no central directory. SimpleX connects people using one-time invites and private relays, keeping your identity out of band.

Updated: 2025-09-10

How it’s different

  • No global ID to track or harvest.
  • Connection via one-time links/QRs; rotate identities as needed.
  • Open-source clients and infrastructure.

Quick start

  1. Install SimpleX on two devices.
  2. Create a new chat identity; share an invite link/QR out-of-band.
  3. Once connected, messages are E2E encrypted; rotate identities for different contexts.

Tips

  • Use different identities for different circles (work/friends/public).
  • Back up your local keys if you move devices.
  • For sensitive use, keep metadata minimal (no cloud backups).
